What is Haskell?

Haskell is a programming language that

  • is pure
  • is lazy
  • has immutable data
  • is used to build programs by composing functions
    • show how it's similar to unix pipes as in this video.

Haskell is Pure

  • This means that functions always return the same output for a given set of inputs, and
  • they have no side effects

Haskell is lazy

  • data is only created as it is needed
