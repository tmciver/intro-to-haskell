What is Haskell?

Haskell is programming language that

  • is pure
  • is lazy
  • has immutable data

Haskell is Pure

  • This means that functions always return the same output for a given set of inputs, and
  • they have no side effects

Haskell is lazy

  • data is only created as it is needed
-- an infinite list
xs = [1..]

-- take the first 10
take 10 xs

Why Haskell is Cool

  • A new way to think about programming
  • expressive, concise and clear
  • beautiful
